Get started with caching

Valkey is an open-source, high-performance key/value datastore that supports workloads such as caching, message queues, and primary databases. It's a community-driven fork and fully compatible with Redis® OSS.

Trusted by the community — 25.6k+ stars on GitHub.

{# Quick Start #}

Quick start

Pick your platform and start running Valkey in seconds.

{# macOS #} 
# Install Valkey
brew install valkey

# Start Valkey as a service
brew services start valkey

# Verify it's running
valkey-cli ping
# → PONG
{# Docker #} 
# Pull and run the latest Valkey image
docker run -d --name valkey -p 6379:6379 valkey/valkey

# Connect with the CLI
docker exec -it valkey valkey-cli ping
# → PONG
{# Linux (apt) #} 
# Debian / Ubuntu / Mint
sudo apt update
sudo apt install valkey

# Start the server
sudo systemctl start valkey-server
sudo systemctl enable valkey-server

# Verify it's running
valkey-cli ping
# → PONG
{# From source #} 
# Download latest release
curl -LO https://github.com/valkey-io/valkey/archive/refs/tags/8.0.2.tar.gz
tar -xzvf 8.0.2.tar.gz
cd valkey-8.0.2

# Build
make
sudo make install

# Run the server
valkey-server

# In another terminal, verify
valkey-cli ping
# → PONG
{# Try It #}

Try it out

Once Valkey is running, open the CLI and try some basic commands.

valkey-cli

# Set a key
SET greeting "Hello, Valkey!"
# → OK

# Get it back
GET greeting
# → "Hello, Valkey!"

# Work with lists
LPUSH tasks "build app" "write docs" "ship it"
LRANGE tasks 0 -1
# → 1) "ship it"  2) "write docs"  3) "build app"

# Set a key with expiration (60 seconds)
SET session:abc123 "user_data" EX 60
TTL session:abc123
# → 59
{# FAQ #}

FAQ

Is it free?

Yes. Valkey is free and open-source under the BSD 3-Clause license. No vendor lock-in, no paid tiers for the core datastore.

Is it compatible with Redis?

Yes. Valkey is a direct fork of Redis OSS 7.2.4 and maintains full protocol and command compatibility. Existing Redis clients, libraries, and tools work with Valkey out of the box.

What can I use Valkey for?

Caching, session storage, real-time leaderboards, rate limiting, pub/sub messaging, message queues (Streams), geospatial indexing, semantic caching, prompt caching for AI/LLM workloads, and more. Anywhere you'd use Redis, you can use Valkey.

Who maintains Valkey?

Valkey is maintained by an open community of contributors and hosted under the Linux Foundation. See the full list of participants.

Can I migrate from Redis?

Yes. Valkey is a drop-in replacement. You can point your existing applications at a Valkey server with no code changes. RDB and AOF persistence files are compatible too.
